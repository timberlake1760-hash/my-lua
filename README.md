# GameSense Lua Scripts

Полный набор скриптов для GameSense (CS:GO) включающий продвинутый Resolver и Custom Aimbot.

## 📋 Содержание

- **resosense_improved_v2.lua** - Улучшенный Resolver с Defensive Double Tap
- **aimbot.lua** - Custom Aimbot с предсказанием и фильтрацией целей

---

## 🎯 Resolver (resosense_improved_v2.lua)

### Возможности:

#### **Типы разрешения:**
- **Default** - Стандартный расчет
- **Jitter** - Чередование углов (+ и -)
- **Alternative** - Вариант Jitter с другой логикой
- **Custom** - Ручная настройка через слайдер Delta
- **Smart** - Адаптивный режим (рекомендуется)

#### **Функции:**
1. **Обнаружение типов движения:**
   - Jumping (прыжок)
   - Landing (приземление)
   - Running (бег)
   - Walking (ходьба)
   - Standing (стояние)

2. **Brute Force (запоминание углов):**
   - Система автоматического запоминания углов противника
   - После 10+ срабатываний использует самый частый угол
   - Индикатор [USING] когда используется запомненный угол

3. **Defensive Double Tap:**
   - Режимы: On Shot, On Damage, Always
   - Настройка по тикам (1-16)
   - Контроль точности (1-100%)

### Рекомендуемые настройки:

### Параметры:

- **Delta** (1-10) - Сила разрешения в Custom режиме
  - Ниже = сильнее
  - Выше = слабее
  - Рекомендуемое значение: 3-4

- **Aggressiveness** (1-100%) - Сила адаптации к движению
  - Рекомендуемое значение: 50-70%

- **DDT Ticks** (1-16) - История позиций врага
  - 8 тиков = оптимально
  - Ниже = агрессивнее, может быть заметно
  - Выше = безопаснее, медленнее

## 🔫 Aimbot (aimbot.lua)

### Возможности:

#### **Режимы:**
- **Silent** - Невидимый aim (изменяет только углы стрельбы)
- **Smooth** - Плавное наведение
- **Flick** - Быстрый флик

#### **Целеполагание:**
- **Target Bone:** Head, Neck, Chest, Best (автоматический выбор)
- **FOV** (1-180°) - Поле зрения обнаружения
- **Prediction** - Предсказание движения врага
- **Prediction Strength** (1-100%) - Сила предсказания

#### **Фильтры:**
- **Only Visible** - Только видимые враги
- **Only Alive** - Только живые враги
- **Skip Teammates** - Игнорировать союзников
- **Min Health** (1-100 HP) - Минимальное здоровье для целеполагания

#### **Продвинутые опции:**
- **Smoothness** (1-100%) - Плавность наведения
- **Dynamic Smoothness** - Адаптивная плавность по дистанции
- **Recoil Control** (0-100%) - Компенсация отдачи
