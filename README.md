# GameSense Lua Scripts

Полный набор скриптов для GameSense (CS:GO) включающий продвинутый Resolver и Custom Aimbot.

## 📋 Содержание

- **resosense_improved_v2.lua** - Улучшенный Resolver с Defensive Double Tap
- **aimbot.lua** - Custom Aimbot с предсказанием и фильтрацией целей

---

## 🎯 Resolver (resosense_improved_v2.lua)

### Возможности:

#### **Типы разрешения:**
- **Default** - Стандартный расчет
- **Jitter** - Чередование углов (+ и -)
- **Alternative** - Вариант Jitter с другой логикой
- **Custom** - Ручная настройка через слайдер Delta
- **Smart** - Адаптивный режим (рекомендуется)

#### **Функции:**
1. **Обнаружение типов движения:**
   - Jumping (прыжок)
   - Landing (приземление)
   - Running (бег)
   - Walking (ходьба)
   - Standing (стояние)

2. **Brute Force (запоминание углов):**
   - Система автоматического запоминания углов противника
   - После 10+ срабатываний использует самый частый угол
   - Индикатор [USING] когда используется запомненный угол

3. **Defensive Double Tap:**
   - Режимы: On Shot, On Damage, Always
   - Настройка по тикам (1-16)
   - Контроль точности (1-100%)
